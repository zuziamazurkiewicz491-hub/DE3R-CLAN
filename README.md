# DE3R CLAN Ticket Bot

## Instalacja
```bash
npm install
```

## Konfiguracja
1. Zmień nazwę `.env.example` na `.env`
2. Wpisz TOKEN, CLIENT_ID i GUILD_ID
3. Odpal bota:

```bash
npm start
```
